## 项目主要需求

### 项目概述

【Web前端】建筑信息模型（BIM）3D 可视化查看器。基于 Three.js 的纯前端建筑模型浏览器，支持楼层切换、构件信息查看、视角管理、标注和剖切分析。数据存储于 localStorage / IndexedDB。**本项目用于验证模型跨文件协同改动能力，禁止将多个职责合并进单一文件。**

### 核心实体（≥ 3 个数据模型）

1. **模型层级（ModelLayer）** — 字段：层ID、层级名称（地基/1F/2F/屋顶/外立面）、可见性、透明度（0-1）、构件数量、层颜色
   - 业务动作：切换可见性、调节透明度、聚焦某层
   - 定义位置：`types/model.ts`

2. **构件信息（BuildingElement）** — 字段：构件ID、所属层ID、名称、类型（Wall/Column/Beam/Slab/Door/Window/Other）、材质、尺寸参数（长×宽×高）、位置坐标（x,y,z）、旋转角度、属性键值对Map
   - 业务动作：点击选中、高亮显示、信息弹窗
   - 定义位置：`types/element.ts`

3. **视角（Viewpoint）** — 字段：视角ID、名称、相机位置（x,y,z）、目标点（x,y,z）、FOV、是否默认视角、创建时间
   - 业务动作：保存当前视角、切换视角、删除视角
   - 定义位置：`types/viewpoint.ts`

4. **标注（Annotation）** — 字段：标注ID、关联构件ID、世界坐标、标注内容、标注类型（问题/说明/变更）、颜色、创建者、创建时间
   - 业务动作：添加标注、编辑内容、删除标注
   - 定义位置：`types/annotation.ts`

5. **剖面（SectionPlane）** — 字段：剖面ID、名称、法向量（x,y,z）、距离原点距离、是否激活
   - 业务动作：创建剖面、调整位置、激活/停用
   - 定义位置：`types/section.ts`

### 核心页面（≥ 4 个）

1. **3D 主视图** `/viewer` — 消费：ModelLayer + BuildingElement + Annotation + SectionPlane
   - 核心 Three.js 渲染区，左侧楼层树，右侧构件信息面板，底部标注列表
   - 使用 `<LayerTree>`, `<AnnotationMarker>`, `<PropertyPanel>` 共享组件

2. **视角管理** `/viewpoints` — 消费：Viewpoint
   - 视角卡片列表，支持保存当前视角、点击切换、设为默认
   - 使用 `<ViewpointCard>` 共享组件

3. **构件浏览器** `/elements` — 消费：BuildingElement + ModelLayer
   - 构件树形列表（按楼层分组），搜索构件，点击跳转到3D视图聚焦
   - 使用 `<PropertyPanel>` 展示构件详情

4. **标注管理** `/annotations` — 消费：Annotation + BuildingElement
   - 标注列表（按类型筛选），点击标注定位到3D视图中的标注位置
   - 使用 `<AnnotationMarker>` 和 `<EmptyState>`

5. **剖面分析** `/sections` — 消费：SectionPlane
   - 剖面列表，支持创建新剖面（交互式拖拽法向量）、激活/停用剖面切片

### 横切关注点

1. **全局异常处理**
   - 触达文件层：Three.js渲染错误捕获 + 数据加载失败fallback + IndexedDB操作异常处理 + 全局ErrorBoundary + 类型定义

2. **主题切换（深色/浅色）**
   - 触达文件层：CSS变量定义 + 主题store + 所有组件主题适配 + Three.js场景背景色切换 + 主题类型定义

### 共享枚举/常量（≥ 2 组）

1. **构件类型枚举 ElementType**：Wall / Column / Beam / Slab / Door / Window / Other
   - 前端：`types/enums.ts`
   - README 必须列出此枚举在所有文件中的出现位置

2. **标注类型枚举 AnnotationType**：Issue / Description / Change
   - 前端：`types/enums.ts`

3. **默认视角常量 DEFAULT_VIEWPOINTS**：俯视 / 正面 / 侧面 / 透视
   - 前端：`constants/viewpoints.ts`

### 共享前端组件清单（≥ 3 个）

1. **`<LayerTree>`** — 消费页面：3D主视图
2. **`<PropertyPanel>`** — 消费页面：3D主视图、构件浏览器
3. **`<AnnotationMarker>`** — 消费页面：3D主视图、标注管理
4. **`<ViewpointCard>`** — 消费页面：视角管理
5. **`<EmptyState>`** — 消费页面：所有列表空状态

### 共享 hooks/utils 清单（≥ 2 个）

1. **`useSceneManager`** — 封装Three.js场景初始化、渲染循环、模型加载逻辑
2. **`useLocalStorage`** — 通用localStorage读写hook，带类型安全
3. **`threeUtils/`** — 包含raycaster封装、坐标转换、材质工厂等工具

### 技术要求

- **前端端口**：18703
- **前端**：React 18 + TypeScript + Vite
- **3D引擎**：Three.js + @react-three/fiber + @react-three/drei
- **UI 库**：Tailwind CSS
- **状态管理**：Zustand
- **本地存储**：IndexedDB（Dexie.js）
- **路由**：React Router v6
- **构建工具**：Vite
